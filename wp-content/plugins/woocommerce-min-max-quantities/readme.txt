=== WooCommerce Min/Max Quantities ===
Contributors: woothemes
Requires at least: 3.1
Tested up to: 3.3
Stable tag: 1.0.2

Lets you define minimum/maximum allowed quantities for products and for orders.

== Usage ==

To set minimum/maximum quantities for products, see the general tab whilst editing a product.

To set minimum/maximum quantities for orders, see the general section in WooCommerce > Settings.

Errors will be displayed on the cart/checkout if a users cart is not valid.
